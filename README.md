
## 从零起一个 webpack + react + mobx 架子

### 如何开始
1. 克隆仓库
```
git clone git@github.com:yinhexidashuaibi/webpack.git
```
2. 下载依赖
```
npm install
```
3. 打包
```
npm run dll:dev

```
4. 启动环境
```
# 启动开发环境
npm run dll:dev
# 构建生成环境资源
npm run build
```

### 特性

- 区分 dev 和 prod 环境
- dll 支持
- mock api 支持
- eslint
